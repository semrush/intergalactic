@## Keyboard support

| Key                                  | Function                                                              |
| ------------------------------------ | --------------------------------------------------------------------- |
| `Space`                              | Selects the focused object and starts dragging.                       |
| Repeated pressing of `Space`         | Terminates dragging.                                                  |
| `Up`, `Down`, `Left`, `Right` arrows | Move the selected object down/up and right/left in the dropping area. |
