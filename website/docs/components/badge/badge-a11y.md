@## What component has

### Roles & attributes

The list below describes roles and attributes that component already has.

| Attribute            | Element | Usage                                                                  |
| -------------------- | ------- | ---------------------------------------------------------------------- |
| `aria-hidden="true"` | `span`  | This element is auxiliary and should not be played by a screen reader. |
