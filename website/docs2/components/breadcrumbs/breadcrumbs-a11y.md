## What component has

### Keyboard support

Table: Keyboard support

| Key              | Function                                         |
| ---------------- | ------------------------------------------------ |
| `Tab`            | Moves focus to the next focusable element.       |
| `Shift + Tab`    | Moves focus to the previous focusable element.   |
| `Space`, `Enter` | When focus is on the breadcrumb, opens the link. |
